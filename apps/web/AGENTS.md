# AGENTS.md

## Web UI Requirements
- Keep the first screen as the operations dashboard, not a marketing page.
- Preserve all navigation areas from `docs/plan.md`.
- Every core metric must have a visible chart, table, or drill-down affordance.
- Critical actions such as rollback, pause release, and cache mode must be auditable by the API.
- Do not render secrets, subscription tokens, WARP private keys, or private node credentials.
